Welcome to GitHub Pages.

这个页面的自动生成是最简单的方法来创建漂亮的页面为您的所有项目。这里作者你的页面内容使用github味的降价，选择一个模板由一个设计师制作和发布。在你的页面生成之后，您可以检查出新的分支

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout gh-pages

如果你使用mac的github，只需同步您的存储库，您将看到新的分支。

Designer Templates

我们已经制作了一些漂亮的模板供您使用。继续继续布局来浏览。您可以轻松地回到编辑在出版之前，你的页面。在发布页面之后，您可以重新访问页面生成器并切换到另一个主题。如果它仍然是降价格式将保留你的页面内容.

Rather Drive Stick?

如果你不喜欢使用自动页面，可以手动命名的一个分支 gh-pages 手动存储库创建页面。除了支持常规的html内容，页面支持jekyll简单，知道静态站点发电机由我们自己写的博客汤姆普雷斯顿。jekyll可以轻松创建站点范围的页眉和页脚，而不必复制他们在每页。它还提供了智能的博客和其他高级模板功能的支持。

Authors and Contributors

You can @mention a GitHub username to generate a link to their profile. The resulting <a> element will link to the contributor's GitHub Profile. For example: In 2007, Chris Wanstrath (@defunkt), PJ Hyett (@pjhyett), and Tom Preston-Werner (@mojombo) founded GitHub.
